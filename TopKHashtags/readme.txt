
1) Change the current directory to "src"

2) The src will have the executable script named "run.sh"

3) Before running the script file make sure XWIN32 is running if the application is ran on VCL

3) Executed the command : " sh run.sh "

4) A GUI will be appear allowing the user to input two values

K =  number of top hashtags to be displayed as the output

Window size = The sliding window size in terms of number of tweets

5)The input values should be of numeric form and numbers greater than zero should be entered. Incase of invalid input user will be prompted with an error msg.

6)After entering the input, the user needs to press the START button to start streaming which will gradually show the required output.

7)In order to stop the streaming the user needs to press the STOP button.
